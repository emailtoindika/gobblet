Project Overview

This project aims to developing a digital version of the Gobblet game. It is a strategic board game identical with the Tic-Tac-Toe game with more complexity. The game use two different size of pieces that can cover smaller pieces, requiring players to remember hidden pieces and think ahead during the game.

The purpose of this project is to create an interactive web base board game that allows users to play online. The target users are who like to play board games and enjoy strategy. The website will allow two players to complete against each out on the same device. Normally, there is no specific duration for making a piece move. However, when timer is enabled, player must make their move within the time duration.

There is a tournament option, and it is deactivated by default. However, when the tournament mode option enabled, the winner will be select after completing the specified number of total games. Game and tournament status can be reset any given time.

The main objective of this project is to design a user-friendly interface and interactive online game. This project uses core web technologies, including HTML, CSS, and JavaScript. In addition, Google sign-in service and browser APIs are used to enhance functionality.
